# 3D Printing

Parametric 3D models for everyday life, designed in OpenSCAD and printed on an Ender 3 S1.

## Projects

- [car/trash_can](car/trash_can/README.md) — Car trash bag hanger that clips over a headrest post.  Note: this melts with PLA lol.
- [cat_router_template](cat_router_template/README.md) — Router and drill jigs for making a cat-shaped wooden box

## Tools

- **CAD:** [OpenSCAD](https://openscad.org/)
- **Printer:** Bambu A1
- **Material:** PLA, 200–210 °C, 0.1–0.28 mm layer height
